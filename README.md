# Araba Alım-Satım ve Kiralama Otomasyonu 🚗

Bu proje, Python programlama dili kullanılarak geliştirilmiş basit bir araç yönetim sistemidir.

## 🚀 Özellikler
- **Araç Kaydı:** Yeni araçları marka, model ve plaka bilgileriyle sisteme ekleme.
- **Envanter Listeleme:** Kayıtlı tüm araçları ve durumlarını görüntüleme.
- **Kiralama Yönetimi:** Araçların kiralama durumlarını takip etme.
- **Satış İşlemleri:** Araç satış simülasyonu.

## 💻 Nasıl Kullanılır?
Terminal üzerinden projeyi başlatmak için:
```bash
python araba.py
